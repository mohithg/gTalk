# gTalk [![Build Status](https://travis-ci.org/tarunbatra/gTalk.svg?branch=master)](https://travis-ci.org/tarunbatra/gTalk)

## Features
* Online/Offline Status
* Web Notifications
* Requesting/Canceling/Accepting/Rejecting requests to chat

## Installation
> npm install

## Usage
> npm start

## Test
> npm test
